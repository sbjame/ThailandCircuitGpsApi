## Getting Started

### Prerequisites
- Node.js (v24 or newer)
- npm or yarn

### Installation
```bash
git clone [https://github.com/sbjame/ThailandCircuitGpsApi.git](https://github.com/sbjame/ThailandCircuitGpsApi.git)
cd ThailandCircuitGpsApi
npm install
```
### Running Locally
```bash
npm run dev
Then open http://localhost:5000 in your browser.
```
